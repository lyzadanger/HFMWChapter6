How to use the Tartan Builder

For each color you'd like to have in your tartan, select the color and the stitch count. Stitch counts can be anywhere from 2 (the smallest a traditional weaver could do in a 2-over-2-under twill weaving procedure) to 72 (this upper limit is somewhat arbitrary). Remember that each stich-and-color combination will be repeated both horizontally and vertically; that's how you get that tartan look!
